# OneReserve — Página web informativa

Repositorio para la página informativa de OneReserve.

Descripción
- Sitio web informativo multilingüe (es / ca) con páginas: Inicio, Servicios, Sobre, Contacto, Aviso Legal y Política de Privacidad.

Ejecución local

Requisitos: Node.js 18+ y npm

Instalar dependencias:

```powershell
npm install
```

Iniciar servidor de desarrollo:

```powershell
npm run dev
```

Construir para producción:

```powershell
npm run build
```

Notas
- El proyecto usa Vite, React + TypeScript y Tailwind CSS.
- Rutas localizadas disponibles en `/es/*` y `/ca/*`.
- El repositorio es privado y está pensado como sitio informativo.
